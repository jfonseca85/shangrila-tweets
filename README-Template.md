# Twitter Streaming Analyze usando Java 8, Spark Streaming, Kafka

Sistema de integração em tempo real que possam manipular dados em escala real com robustez e tolerância a falhas usando ferramentas como Apache Spark, Kafka, Cassandra e Hadoop. Utilizando um design inteligente big data  permite combinar dados de fluxo de baixa latência em cargas de trabalho em aplicativos a usando o Apache Spark, o Spark DataFrames e o Spark SQL, além da API de fontes de dados do Spark para carregar, armazenar e manipular integração de Spark Streaming Spark-Kafka para confiabilidade e velocidade.
